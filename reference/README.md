# References

The prototype was developed through study of the following publications:

- Sergio Luque. “Stochastic Synthesis: Origins and Extensions.” 2006.
- Peter Hoffmann. “The Genesis of GENDY3.” 2022.

These publications describe the distinction between the GENDY signal process and the PARAG formal architecture, bounded second-order random walks, exponential time fields, Bernoulli silences, multitrack organisation, and the pseudo-stereo procedure associated with *GENDY3*.

The publications themselves are not redistributed here. Consult the original publishers or authorised academic sources.
